This takes about 10 minutes. You'll need Zoom admin access.

  1. Go to marketplace.zoom.us and sign in with your Zoom admin account
  2. Click Develop in the top nav, then Build App
  3. You'll see several app types. Choose Server-to-Server OAuth (not "Webhook Only" or "OAuth")
    Common mistake: "Webhook Only" apps won't work because they can't pull recordings
  4. Name the app "Fairplay" (or any name you prefer) and click Create
  5. App Credentials page: You'll see Account ID, Client ID, and Client Secret. Copy all three and paste them in the fields above
    Tip: Keep this page open, you'll come back to it
  6. Information page: Fill in your email under Developer Contact. You can skip the Privacy Policy, Terms, and Support URLs. Click Continue
  7. Feature page: You'll see a Secret Token and Event Subscriptions toggle. Leave Event Subscriptions off for now. Click Continue
    If it asks for an Event Notification URL, just skip it. We don't need webhooks.
  8. Scopes page: Click + Add Scopes. Search for "recording" and add these two:
    • cloud_recording:read:list_user_recordings (under "View all user recordings")
    • cloud_recording:read:meeting_transcript (under "Read meeting transcript")
    Click Done, then Continue
  9. Activation page: Click Activate your app. If it says you're missing something, check that both scopes were added

Also make sure cloud recording and audio transcript are turned on in your Zoom account settings (Settings > Recording > Cloud recording + Audio transcript). Most accounts have this on already.

How do I get this?
  1. Go to app.attio.com and log in
  2. Click Settings (gear icon, bottom left)
  3. Go to Developers > API Keys
  4. Click Generate new API key
  5. Give it read/write access to Deals, Companies, and People
  6. Copy the key and paste it above

You'll need HubSpot admin access for this.

  1. Go to app.hubspot.com and log in
  2. Click Settings (gear icon, top right)
  3. Go to Integrations > Private Apps
    Note: HubSpot recently moved Private Apps to "Legacy Apps." If you see that page, click "Create legacy app" or use an existing one.
  4. Click Create a private app (or "Create legacy app") and name it "Fairplay"
  5. Under Scopes, add these four:
    • crm.objects.deals.write
    • crm.objects.deals.read
    • crm.objects.contacts.read
    • crm.objects.companies.read
  6. Click Create app, then copy the Access token (starts with "pat-")
  7. Paste the token above

If you already have a Private/Legacy App with deal + contact + company access, you can reuse that token instead of creating a new one.

Get notified in Slack when calls are scored.
How do I set this up?
  1. Go to api.slack.com/apps and sign in to your workspace
  2. Click Create New App > From scratch
  3. Name it "Fairplay" and select your workspace
  4. Go to Incoming Webhooks in the left sidebar
  5. Toggle Activate Incoming Webhooks to On
  6. Click Add New Webhook to Workspace
  7. Select the channel where you want notifications (e.g. #deals)
  8. Copy the Webhook URL and paste it above

Already have a Slack app? Just add an Incoming Webhook to it from the app settings.
